About Smart Contact Manager

Smart Contact Manager is a modern, feature-rich application designed to make managing contacts simple and efficient. Built using Java, Spring Boot, Spring MVC, Spring Data JPA, Spring Security, HTML, CSS, JavaScript, Bootstrap, and a MySQL database, this application offers a seamless experience for users.

After successful registration, all user details are securely stored in the database, allowing users to log in and access the system with ease.

Once logged in, users can add, view, edit, and delete contacts, manage their profiles, update passwords, and organize their contact list efficiently.
The application also includes a Favorites feature, allowing users to mark and quickly access important contacts whenever needed.
